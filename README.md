# **Tiepolo** 

## 概要
**Tiepolo（ティエポロ）** は、従来のペイントツールが持つ「歴史的な慣習」をゼロベースで検証し、現代の描き手の視点から再構築した新世代のペイントツールです。

ペイントツール黎明期から受け継がれてきた「デジタルペイントの標準的な仕様」に縛られることなく、イラストレーターのワークフローを最優先に考えた機能群を搭載しています。直感的なレイヤー管理や人間の知覚に寄り添ったカラーサークルなど、初心者でも迷わず描ける優れた操作性を持ちながら、GLSLによるカスタムシェーダーや高度なカラーキャリブレーション機能など、上級者のこだわりに応える強力な拡張性も兼ね備えています。

## 独自機能
#### クリッピングのスタックとインジケーター
クリッピングマスクを横方向にスタックして配置するUIを採用。これにより、レイヤー数が増加しても依存関係が明確になります。
  
また、全レイヤーを俯瞰できる「レイヤーインジケーター」を常に表示。クリッピングを「ブランチ（枝）」、ブレンドモードを「色」、レイヤーの種類を「形状」で視覚化することで、複雑な構造も一目で把握できます。インジケーター上をドラッグしてレイヤーを移動できるため、レイヤー整理の操作も劇的にスムーズになります。

<img width="411" height="630" alt="image" src="https://github.com/user-attachments/assets/11d50739-b13b-4797-aa37-f54196574fc1" />


#### 色相シフトを備えたOKLCHカラーサークル
人間の知覚に基づいて色を選択できる「OKLCH色空間」を採用。その特殊な形状ゆえにUIへの落とし込みが難しかった点を、Tiepolo独自の設計で解決しました。RGBでは表現できない領域に「別の色相」を最適に割り当てることで、「光の色は黄色寄り」「影の色は青寄り」といったデジタルイラスト特有の色選びが、カラーサークルを回転させることなく矩形内で完結します。

<img width="417" height="401" alt="image" src="https://github.com/user-attachments/assets/9b7cf65c-4118-4c1b-84e8-f8afbc26c372" />


#### キャンバスのループ機能
キャンバスの端を上下・左右でシームレスに繋げるループ機能を搭載。左端を越えたストロークがそのまま右端へリアルタイムに描画されるため、タイリング可能なテクスチャの作成が容易になります。変形ツールを使用してテクスチャ全体をにスクロールさせながら描画することも可能です。

<img width="1014" height="521" alt="image" src="https://github.com/user-attachments/assets/007b0c98-f93e-4add-8659-361ac5e0b88f" />


#### 自由にGLSLを書けるカスタムシェーダー
GLSL（OpenGL Shading Language）を用いて、元の色から変換後の色を計算する独自のプログラムを組み込めます。手描きでは困難な精密なグラデーションや幾何学パターンの生成など、テクスチャ作成を強力にサポート。アイデア次第で、オリジナルの画像編集エフェクトやフィルターを自作することも可能です。

<img width="1116" height="847" alt="image" src="https://github.com/user-attachments/assets/894e4319-d513-4199-bf42-f57f83cae1bf" />


#### 表示色を変えられるカラープレビューとキャリブレーション
内部データはRGBで保持しつつ、表示モードを「RGB」「CMYK」「グレースケール（輝度）」「グレースケール（明度）」の4種類から即座に切り替え可能。「明度ベースでの陰影バランスをモノクロで確認したい」といった検証がワンクリックで行えます。

さらに、標準でカラーキャリブレーション機能を内蔵。PCモニター、液晶タブレット、スマートフォンなど、デバイス間で発生する色味の差異を、OSの複雑な設定を開くことなく、実際のキャンバスを見ながらアプリ内で直感的に補正できます。

これらの表示設定は、キャンバスだけでなくカラーサークルにも適用されます。

#### ショートカットキー
デフォルトのキーバインドは、すべての主要アクションが左手だけで完結する範囲に最適化されています。もちろん、ユーザーのプレイスタイルに合わせた自由なカスタマイズにも対応しています。
