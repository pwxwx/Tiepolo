# ショートカット

ショートカットは、初期設定ではキーボードの左手で基本操作ができるよう配置しています。設定意図を記載するので、覚える際にお役立てください。

また、`上部メニューバー > 設定 > ショートカットキー設定`から変更することができます。

### ツール
原則、英語のツール名の頭文字を設定しています。
|ツール|ショートカットキー|設定理由|
|---|---|---|
|ペン|Q| 英語で羽根ペンを表すQuillから。Pなどは左手で押せないため。|
|消しゴム|E| Eraser|
|エアブラシ|A| AirBrush|
|塗りつぶし|F|Fill|
|スポイト|D|Dropper|
|選択|S|Select|
|ぼかし|B|Blur|
|ゆがみ|W|Warp|
|テキスト|T|Text|
|移動|Space| 使用頻度が高いため。一般的なペイントツールの設定に倣った。|
|回転|R|Rotate|
